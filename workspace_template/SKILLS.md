# SKILLS.md — Capacidades Disponibles

> Índice de skills instalados en este workspace.
> Para usar un skill: lee `skills/[nombre]/SKILL.md` primero.

## Skills Instalados

### browser
- **Descripción:** Control de Google Chrome vía CDP (Chrome DevTools Protocol)
- **Capacidades:** Navegar URLs, leer HTML/JS, ejecutar código en consola, login en cuentas
- **Instrucciones:** `skills/browser/SKILL.md`

### gog
- **Descripción:** Google Workspace — Gmail, Calendar, Drive, Contacts, Sheets, Docs
- **Capacidades:** Leer y enviar correos, gestionar calendario, buscar en Drive, editar Sheets
- **Instrucciones:** `skills/gog/SKILL.md`

---

## Agregar un Nuevo Skill

1. Crea la carpeta `skills/[nombre]/`
2. Escribe `skills/[nombre]/SKILL.md` con instrucciones de uso para Claude
3. Agrega cualquier script de soporte necesario en esa carpeta
4. Actualiza este archivo con la nueva entrada
